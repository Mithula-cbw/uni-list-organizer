# Uni List Organi**Z**er

A lightweight, local-first tool for ranking university course 
applications by confidence level. Built for Sri Lankan A/L 
students who are managing multiple course choices and want a 
clear, structured way to prioritize before decisions lock in.

No backend. No account. Data stays in your browser.

## What it does

- Add courses with university, Z-score cutoff, and confidence level
- Drag to reorder and rank by preference
- Lock entries to protect finalized choices from accidental reordering
- Filter and search across your list
- Export/import as JSON for backup or sharing

## Use it

<pre>
  ⚠️ This tool is NOT affiliated with the UGC or any university. Data you enter
  is not validated by us — confirm all cutoffs directly with official sources.
</pre>

👉🏻 visit the live version: [[Uni List OrganiZer](https://mithula-cbw.github.io/uni-list-organizer/)]

## Why this exists

Tracking a course list with 125+ options across spreadsheets and notes gets 
messy fast. This gives it structure without the overhead.
